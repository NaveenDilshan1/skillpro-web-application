# SkillPro Institute Web Application

## 📌 Project Overview
SkillPro Institute is a TVEC-registered vocational training center in Sri Lanka.  
This system is developed to manage courses, students, instructors, and enrollments.

## 🚀 Features
- User Registration & Login (Admin / Instructor / Student)
- Course Management System
- Student Dashboard
- Instructor Dashboard
- Admin Panel
- Course Enrollment
- Event Calendar
- Payment Management
- Feedback System

## 🛠️ Technologies Used
- Frontend: HTML, CSS, Bootstrap, React.js
- Backend: PHP (Laravel)
- Database: MySQL

## 🧩 System Modules
- Authentication System
- Course Management
- Enrollment System
- Dashboard System
